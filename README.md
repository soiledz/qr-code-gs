# QR Code Generator

This project is a QR code generator built with React and TypeScript.

## Features

- Generate QR codes with customizable size and error correction level
- Save generated QR codes to the local storage
- Display the generated QR code and the input text in the UI

## Prerequisites

Before running this project, make sure you have the following installed:

- Node.js
- npm or yarn

## Installation

1. Clone the repository:

   ```shell
   git clone [https://github.com/your-username/qr-code-generator.git](https://github.com/your-username/qr-code-generator.git)

2. Install the dependencies:

   ```shell
   cd qr-code-generator
   npm install
   ```

3. Start the development server:

   ```shell
   npm run dev
   ```

4. Open your browser and navigate to [http://localhost:5173](http://localhost:5173)

## Usage

1. Enter the text you want to generate a QR code for in the input field
2. Adjust the size and error correction level of the QR code
3. Click the "Generate QR Code" button
4. The generated QR code will be displayed in the UI
5. Click the "Save QR Code" button to save the QR code to the local storage
6. Reload the page to see the saved QR codes
